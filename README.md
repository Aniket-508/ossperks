## AI & Machine Learning

- [Anthropic Claude](https://claude.com/contact-sales/claude-for-oss) - Six months of free Claude Max 20x for open-source maintainers and contributors.
- [CodeRabbit](https://www.coderabbit.ai/blog/coderabbit-commits-1-million-to-open-source) - Free AI-driven code reviews for public repositories on GitHub and GitLab.
- [Codex Open Source Fund](https://openai.com/form/codex-open-source-fund/) - One-time API credit grants of up to $25,000 for projects integrating Codex CLI or OpenAI models.
- [GitHub Copilot](https://docs.github.com/en/copilot/managing-copilot/managing-copilot-as-an-individual-subscriber/managing-your-github-copilot-pro-subscription/getting-free-access-to-copilot-pro-as-a-student-teacher-or-maintainer) - Free Copilot Pro for maintainers of popular open-source repositories.
- [Greptile](https://www.greptile.com/open-source) - Free AI code review for OSS projects with MIT, Apache, or GPL licenses.
- [OpenAI Codex](https://openai.com/form/codex-for-oss/) - Six months of free ChatGPT Pro with Codex access and API credits for maintainers.
- [Sourcery](https://sourcery.ai/pricing) - Free Pro-tier AI code reviews for public repositories with IDE integration and security scans.

## Analytics

- [OpenPanel](https://openpanel.dev/open-source) - Free analytics with up to 2.5 million events per month and full feature access.

## CI/CD

- [Blacksmith](https://www.blacksmith.sh/pricing#oss) - Free high-performance GitHub Actions runners with 2x faster hardware and 4x faster caching.
- [CircleCI](https://circleci.com/open-source/) - Free build credits covering Linux, Arm, Docker, macOS, and Windows.
- [Semaphore](https://semaphore.io/open-source) - Unlimited CI/CD minutes for OSS with four parallel Linux and one macOS machine.

## Communication

- [Zulip](https://zulip.com/for/open-source/) - Free Zulip Cloud Standard with organized topic-based team chat and full message history.

## Credentials & Secrets

- [1Password](https://github.com/1Password/for-open-source) - Free Teams account for secure storage of credentials, API tokens, and secrets.

## Developer Tools

- [Algolia](https://www.algolia.com/for-open-source) - Free or discounted search plans with 200K records and 200K requests per month.
- [Atlassian](https://www.atlassian.com/software/views/open-source-license-request) - Free Standard cloud products including Jira, Confluence, Trello, and Bitbucket.
- [Cal.com](https://cal.com/help/event-types/sponsor) - Free scheduling platform access for OSS projects in exchange for a README and website banner.
- [Crowdin](https://crowdin.com/page/open-source-project-setup-request) - Free localization platform for OSS with translation memory, glossaries, and 700+ integrations.
- [GitBook](https://docs.gitbook.com/account-management/plans/apply-for-the-non-profit-open-source-plan) - Free Community plan with all Ultimate features, GitHub/GitLab sync, and unlimited contributors.
- [GitLab](https://about.gitlab.com/solutions/open-source/) - Free Ultimate-tier features and 50,000 compute minutes per month.
- [JetBrains](https://www.jetbrains.com/community/opensource/) - Free All Products Pack IDE subscriptions for core project teams.
- [Mintlify](https://www.mintlify.com/oss-program) - Free Pro plan for non-commercial OSS projects (custom domains, analytics, API playground, AI chat).
- [Sanity](https://www.sanity.io/docs/open-source-plan) - Free Growth plan for non-monetized OSS projects with 3 datasets, 25 users, and SAML SSO.

## Funding

- [Microsoft FOSS Fund](https://github.com/microsoft/foss-fund) - Quarterly sponsorship of up to $12,500 for open source projects used and nominated by Microsoft employees.

## Hosting & Deployment

- [Netlify](https://www.netlify.com/open-source/) - Dedicated open-source plan with generous credits, unlimited team members, and free deploys.
- [Vercel](https://vercel.com/open-source-program) - Platform credits and an OSS Starter Pack with third-party credits.

## Infrastructure

- [Cloudflare](https://www.cloudflare.com/en-gb/lp/project-alexandria/) - Annual credits for Zero Trust, R2, Workers, Pages, and plan upgrades via Project Alexandria.
- [Convex](https://www.convex.dev/open-source-program) - Free Pro credits for developers building open-source tools and frameworks.
- [DigitalOcean](https://digitalocean.com/open-source/credits-for-projects) - Annual credits tiered by GitHub stars, from $60/year (100+ stars) to $20,000/year (10K+ stars).
- [Docker](https://docs.docker.com/docker-hub/repos/manage/trusted-content/dsos-program/) - Verified badges, rate limit removal, Docker Scout access, and image analytics via the DSOS Program.
- [Neon](https://neon.com/programs/open-source) - Up to $5,000 in annual PostgreSQL credits plus cash referral payouts for OSS projects.
- [Upstash](https://upstash.com/open-source) - Up to $1,000 in monthly credits covering Redis, Kafka, and QStash, plus direct engineering support.

## Monitoring & Observability

- [Datadog](https://www.datadoghq.com/partner/open-source/) - Free observability and security monitoring for OSS projects, used by Apache, Python, and Electron.
- [Sentry](https://sentry.io/for/open-source/) - Free error tracking and performance monitoring with generous usage limits.

## Security

- [Codacy](https://www.codacy.com/pricing) - Free Team plan for OSS with PR integration, quality gates, and security scanning across 49 languages.
- [Code Climate](https://codeclimate.com/quality/pricing) - Free forever code coverage, maintainability analysis, and linting for all public repos.
- [DeepSource](https://deepsource.com/pricing) - Free static analysis, SAST, and automated code reviews with unlimited public repos and members.
- [SignPath](https://signpath.io/solutions/open-source-community) - Free code signing certificate with secure signing and full audit trails.
- [Snyk](https://snyk.io/open-source/) - Full Enterprise access with unlimited vulnerability scanning and dependency analysis.
- [SonarCloud](https://www.sonarsource.com/open-source-editions/sonarqube-cloud/) - Free static analysis with unlimited members and feature branch analysis.

## Testing

- [Argos](https://argos-ci.com/docs/open-source) - Free visual regression testing for OSS projects, with integrations for Playwright, Storybook, Cypress, and more.
- [BrowserStack](https://www.browserstack.com/open-source) - Lifetime access to cross-browser, real device, and visual testing platform.
- [Chromatic](https://chromatic.com/docs/open-source) - 35,000 free Chrome snapshots per month for qualifying OSS projects, by the Storybook team.
- [Codecov](https://about.codecov.io/for/open-source/) - Free forever code coverage with unlimited public repos, PR comments, and status checks.
- [Coveralls](https://coveralls.io/) - Free forever code coverage tracking with multi-language support and unlimited users.
