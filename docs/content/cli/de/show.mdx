---
title: ossperks show
description: Vollständige Details für ein bestimmtes OSS-Vorteilsprogramm anzeigen.
---

## Verwendung

```
ossperks show <slug> [optionen]
```

## Argumente

| Argument | Beschreibung                                       |
| -------- | -------------------------------------------------- |
| `<slug>` | Der Programm-Slug (z. B. `vercel`, `github-pages`) |

## Optionen

| Flag         | Beschreibung        |
| ------------ | ------------------- |
| `--json`     | Rohes JSON ausgeben |
| `-h, --help` | Hilfe anzeigen      |

## Beispiele

```bash

# Das Vercel-Programm anzeigen

ossperks show vercel

# Maschinenlesbares JSON

ossperks show vercel --json
```

## Ausgabe

```
─────────────────────────────────────
  Vercel for Open Source
  von Vercel  ·  Hosting
─────────────────────────────────────

Vercel stellt Plattform-Guthaben und zusätzliche Dienste für
Open-Source-Projekte bereit, die auf der Vercel-Plattform gehostet werden.

VORTEILE
  • 3.600 $ Plattform-Guthaben — 3.600 $ an Vercel-Plattform-Guthaben,
    verteilt über 12 Monate.
  • OSS Starter Pack — Guthaben von Drittanbieter-Diensten.

VORAUSSETZUNGEN
  • Muss ein Open-Source-Projekt sein, das aktiv entwickelt und gepflegt wird.
  • Muss auf Vercel gehostet sein (oder zur Nutzung von Vercel vorgesehen sein).

WIE BEWERBEN
  • Bewerben Sie sich über die Vercel OSS-Programmseite, sobald Bewerbungen möglich sind.

  → Jetzt bewerben: https://vercel.com/open-source-program
```
