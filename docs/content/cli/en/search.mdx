---
title: ossperks search
description: Search OSS perk programs by keyword across names, descriptions, providers, tags, and perks.
---

## Usage

```
ossperks search [query] [options]
```

**Aliases:** `ossperks find`, `ossperks s`

## Arguments

| Argument  | Description                                                                       |
| --------- | --------------------------------------------------------------------------------- |
| `[query]` | Search term (matches name, id, description, provider, category, tags, or perks). Omit in a TTY for interactive autocomplete. |

## Options

| Flag         | Description     |
| ------------ | --------------- |
| `--json`     | Output raw JSON |
| `-h, --help` | Show help       |

## Examples

```bash
# Search for "hosting"
ossperks search hosting

# Interactive autocomplete search (TTY): type to filter, pick a program
ossperks search

# Using the short alias
ossperks s credits

# JSON output
ossperks search vercel --json
```
