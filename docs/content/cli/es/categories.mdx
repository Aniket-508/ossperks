---
title: ossperks categories
description: Lista todas las categorías de programas disponibles y sus slugs.
---

## Uso

```
ossperks categories [opciones]
```

## Opciones

| Opción       | Descripción             |
| ------------ | ----------------------- |
| `--json`     | Genera JSON sin formato |
| `-h, --help` | Muestra ayuda           |

## Ejemplos

```bash

# Listar todas las categorías

ossperks categories

# Salida en JSON

ossperks categories --json
```

## Salida

```
CATEGORY SLUG      LABEL
hosting            Hosting & Deployment
devtools           Developer Tools
security           Security
monitoring         Monitoring & Observability
communication      Communication
ci-cd              CI/CD
testing            Testing
ai                 AI & Machine Learning
analytics          Analytics
infrastructure     Infrastructure
credentials        Credentials & Secrets
```
