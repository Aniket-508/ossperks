import { DynamicLink } from "fumadocs-core/dynamic-link";

## Uso

```
ossperks categories [options]
```

**Alias:** `ossperks cats`

## Opciones

| Flag         | Descripción         |
| ------------ | ------------------- |
| `--json`     | Mostrar JSON crudo  |
| `-h, --help` | Mostrar ayuda       |

## Ejemplos

```bash

# TTY: autocompletar una categoría, luego listar programas (NOMBRE + URL).

# Non-TTY: imprime una tabla de categorías (NOMBRE + CANTIDAD).

ossperks categories

# JSON: ids de categorías para ossperks list --category <id>

ossperks categories --json
```

## Salida

En un **TTY**, obtienes un prompt de autocompletado; después de elegir una categoría, la CLI imprime una tabla de **NOMBRE** y **URL** para los programas en esa categoría (mismo estilo que <DynamicLink href="/[lang]/cli/list">`ossperks list`</DynamicLink>).

En un entorno **no interactivo**:

```
13 categorías
─────────────

  NOMBRE                  CANTIDAD
  Hosting & Deployment       12
  Developer Tools             8
  ...
```

La columna **CANTIDAD** es el número de programas en esa categoría (valores de ejemplo).

Usa el **id** de categoría de `--json` con <DynamicLink href="/[lang]/cli/list">`ossperks list --category <id>`</DynamicLink> para filtrar programas en scripts.
