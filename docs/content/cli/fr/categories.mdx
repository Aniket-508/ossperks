---
title: ossperks categories
description: Lister toutes les catégories de programmes disponibles et leurs slugs.
---

## Utilisation

```
ossperks categories [options]
```

## Options

| Drapeau      | Description           |
| ------------ | --------------------- |
| `--json`     | Afficher le JSON brut |
| `-h, --help` | Afficher l'aide       |

## Exemples

```bash

# Lister toutes les catégories

ossperks categories

# Sortie JSON

ossperks categories --json
```

## Sortie

```
CATEGORY SLUG      LABEL
hosting            Hébergement & Déploiement
devtools           Outils de Développement
security           Sécurité
monitoring         Surveillance & Observabilité
communication      Communication
ci-cd              CI/CD
testing            Tests
ai                 IA & Apprentissage Automatique
analytics          Analytique
infrastructure     Infrastructure
credentials        Identifiants & Secrets
```
