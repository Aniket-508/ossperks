## Utilisation

```
ossperks categories [options]
```

**Alias :** `ossperks cats`

## Options

| Flag         | Description           |
| ------------ | --------------------- |
| `--json`     | Afficher le JSON brut |
| `-h, --help` | Afficher l'aide       |

## Exemples

```bash

# TTY : autocomplétion d'une catégorie, puis liste des programmes (NOM + URL).

# Non-TTY : affiche un tableau des catégories (NOM + NOMBRE).

ossperks categories

# JSON : identifiants de catégories pour ossperks list --category <id>

ossperks categories --json
```

## Résultat

Dans un **TTY**, vous obtenez une invite d'autocomplétion ; après avoir sélectionné une catégorie, le CLI affiche un tableau **NOM** et **URL** pour les programmes de cette catégorie (même style que <DynamicLink href="/[lang]/cli/list">`ossperks list`</DynamicLink>).

Dans un environnement **non interactif** :

```
13 catégories
─────────────

  NOM                     NOMBRE
  Hosting & Deployment        12
  Developer Tools              8
  ...
```

La colonne **NOMBRE** indique le nombre de programmes dans cette catégorie (valeurs d'exemple).

Utilisez l'**id** de catégorie issu de `--json` avec <DynamicLink href="/[lang]/cli/list">`ossperks list --category <id>`</DynamicLink> pour filtrer les programmes dans vos scripts.
