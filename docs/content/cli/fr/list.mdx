---
title: ossperks list
description: Lister tous les programmes d'avantages OSS disponibles, avec
  filtrage optionnel par catégorie.
---

## Utilisation

```
ossperks list [options]
```

**Alias :** `ossperks ls`

## Options

| Drapeau                 | Description                              |
| ----------------------- | ---------------------------------------- |
| `-c, --category <id>` | Filtrer par identifiant de catégorie (par ex. `hosting`) |
| `--json`                | Afficher le JSON brut                    |
| `-h, --help`            | Afficher l'aide                          |

## Exemples

```bash

# Lister tous les programmes

ossperks list

# Filtrer uniquement les programmes d'hébergement

ossperks list --category hosting

# Sortie JSON lisible par machine

ossperks list --json
```
