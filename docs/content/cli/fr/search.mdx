---
title: ossperks search
description: Recherchez des programmes d'avantages OSS par mot-clé parmi les
  noms, descriptions, fournisseurs, tags et avantages.
---

## Utilisation

```
ossperks search [requête] [options]
```

**Alias :** `ossperks find`, `ossperks s`

## Arguments

| Argument    | Description                                                                       |
| ----------- | --------------------------------------------------------------------------------- |
| `[requête]` | Terme de recherche (correspond au nom, id, description, fournisseur, catégorie, tags ou avantages). Omettre dans un TTY pour l'autocomplétion interactive. |

## Options

| Option       | Description       |
| ------------ | ----------------- |
| `--json`     | Sortie JSON brute |
| `-h, --help` | Afficher l'aide   |

## Exemples

```bash

# Rechercher « hosting »

ossperks search hosting

# Recherche interactive par autocomplétion (TTY) : tapez pour filtrer, sélectionnez un programme

ossperks search

# Utilisation de l'alias court

ossperks s credits

# Sortie JSON

ossperks search vercel --json
```
