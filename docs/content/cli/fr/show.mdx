---
title: ossperks show
description: Afficher les détails complets d'un programme de bénéfices OSS spécifique.
---

## Utilisation

```
ossperks show [id] [options]
```

**Alias :** `ossperks info`

## Arguments

| Argument | Description                                                                                                             |
| -------- | ----------------------------------------------------------------------------------------------------------------------- |
| `[id]`   | Identifiant de programme optionnel (par ex. `vercel`, `github-pages`). Omettez dans un TTY pour choisir dans une liste. |

## Options

| Drapeau      | Description           |
| ------------ | --------------------- |
| `--json`     | Afficher le JSON brut |
| `-h, --help` | Afficher l'aide       |

## Exemples

```bash

# Afficher le programme Vercel

ossperks show vercel

# Sélecteur interactif (TTY) : choisissez un programme dans la liste

ossperks show

# JSON lisible par machine

ossperks show vercel --json
```
