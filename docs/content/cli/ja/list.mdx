---
title: ossperks list
description: 利用可能なすべてのOSS特典プログラムを一覧表示し、オプションでカテゴリでフィルタリングします。
---

## 使用方法

```
ossperks list [options]
```

**エイリアス:** `ossperks ls`

## オプション

| フラグ                | 説明                                       |
| --------------------- | ------------------------------------------ |
| `-c, --category <id>` | カテゴリIDでフィルタリング (例: `hosting`) |
| `--json`              | 生のJSONを出力                             |
| `-h, --help`          | ヘルプを表示                               |

## 使用例

```bash

# すべてのプログラムを一覧表示

ossperks list

# ホスティングプログラムのみにフィルター

ossperks list --category hosting

# 機械可読なJSON出力

ossperks list --json
```

## 出力

```
48 programs
────────────

  NAME                                          URL
  1Password for Open Source Projects            https://ossperks.com/programs/1password
  Algolia for Open Source                         https://ossperks.com/programs/algolia
  Anthropic Claude for Open Source                https://ossperks.com/programs/anthropic-claude
  ...
```
