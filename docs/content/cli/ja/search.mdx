title: ossperks search
description: 名前、説明、プロバイダーを横断してキーワードでOSSパークプログラムを検索します。
---

## 使用方法

```
ossperks search <query> [options]
```

## 引数

| 引数      | 説明                                                 |
| --------- | ---------------------------------------------------- |
| `<query>` | 検索キーワード(名前、説明、またはプロバイダーに一致) |

## オプション

| フラグ                  | 説明                         |
| ----------------------- | ---------------------------- |
| `-c, --category <slug>` | 特定のカテゴリーに検索を限定 |
| `--json`                | 生のJSONを出力               |
| `-h, --help`            | ヘルプを表示                 |

## 使用例

```bash

# 「hosting」を検索

ossperks search hosting

# devtoolsカテゴリー内で「credits」を検索

ossperks search credits --category devtools

# JSON出力

ossperks search vercel --json
```
