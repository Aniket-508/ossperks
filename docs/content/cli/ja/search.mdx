## 使用方法

```
ossperks search [query] [options]
```

**エイリアス:** `ossperks find`、`ossperks s`

## 引数

| 引数  | 説明                                                                       |
| --------- | --------------------------------------------------------------------------------- |
| `[query]` | 検索語句(名前、ID、説明、プロバイダー、カテゴリ、タグ、または特典に一致)。TTYでは省略すると対話型オートコンプリートが使用できます。 |

## オプション

| フラグ                  | 説明                         |
| ----------------------- | ---------------------------- |
| `-c, --category <slug>` | 特定のカテゴリーに検索を限定 |
| `--json`                | 生のJSONを出力               |
| `-h, --help`            | ヘルプを表示                 |

## 使用例

```bash

# 「hosting」を検索

ossperks search hosting

# 対話型オートコンプリート検索(TTY):入力してフィルタリングし、プログラムを選択

ossperks search

# 短縮エイリアスの使用

ossperks s credits

# JSON出力

ossperks search vercel --json
```
