---
title: ossperks show
description: 特定のOSS特典プログラムの詳細を表示します。
---

## 使用方法

```
ossperks show <slug> [options]
```

## 引数

| 引数     | 説明                                               |
| -------- | -------------------------------------------------- |
| `<slug>` | プログラムスラッグ（例：`vercel`、`github-pages`） |

## オプション

| フラグ       | 説明           |
| ------------ | -------------- |
| `--json`     | 生のJSONを出力 |
| `-h, --help` | ヘルプを表示   |

## 例

```bash

# Vercelプログラムを表示

ossperks show vercel

# 機械可読なJSON

ossperks show vercel --json
```

## 出力

```
─────────────────────────────────────
  Vercel for Open Source
  by Vercel  ·  hosting
─────────────────────────────────────

Vercelは、Vercelプラットフォーム上でホストされている
オープンソースプロジェクトにプラットフォームクレジットと
追加サービスを提供します。

特典
  • $3,600のプラットフォームクレジット — 12ヶ月間にわたって配布される
    $3,600相当のVercelプラットフォームクレジット。
  • OSSスターターパック — サードパーティサービスからのクレジット。

応募資格
  • 積極的に開発・維持されているオープンソースプロジェクトであること。
  • Vercel上でホストされている(またはホストする予定である)こと。

応募方法
  • 応募受付期間中にVercel OSSプログラムページから応募してください。

  → 今すぐ応募: https://vercel.com/open-source-program
```
