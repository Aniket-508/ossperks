## 사용법

```
ossperks categories [options]
```

## 옵션

| 플래그       | 설명           |
| ------------ | -------------- |
| `--json`     | 원시 JSON 출력 |
| `-h, --help` | 도움말 표시    |

## 예제

```bash

# 모든 카테고리 나열

ossperks categories

# JSON 출력

ossperks categories --json
```

## 출력

```
CATEGORY SLUG      LABEL
hosting            호스팅 및 배포
devtools           개발자 도구
security           보안
monitoring         모니터링 및 관찰성
communication      커뮤니케이션
ci-cd              CI/CD
testing            테스트
ai                 AI 및 머신 러닝
analytics          분석
infrastructure     인프라
credentials        인증 정보 및 시크릿
```
