## 사용법

```
ossperks search [query] [options]
```

**별칭:** `ossperks find`, `ossperks s`

## 인수

| 인수  | 설명                                                                       |
| --------- | --------------------------------------------------------------------------------- |
| `[query]` | 검색어 (이름, ID, 설명, 제공자, 카테고리, 태그 또는 혜택과 일치). TTY에서 대화형 자동완성을 위해 생략 가능. |

## 옵션

| 플래그       | 설명           |
| ------------ | -------------- |
| `--json`     | 원시 JSON 출력 |
| `-h, --help` | 도움말 표시    |

## 예제

```bash

# "hosting" 검색

ossperks search hosting

# 대화형 자동완성 검색 (TTY): 입력하여 필터링하고 프로그램 선택

ossperks search

# 짧은 별칭 사용

ossperks s credits

# JSON 출력

ossperks search vercel --json
```

## 출력
