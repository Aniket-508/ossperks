---
title: ossperks show
description: 특정 OSS 혜택 프로그램의 전체 세부 정보를 표시합니다.
---

## 사용법

```
ossperks show <slug> [options]
```

## 인수

| 인수     | 설명                                           |
| -------- | ---------------------------------------------- |
| `<slug>` | 프로그램 슬러그 (예: `vercel`, `github-pages`) |

## 옵션

| 플래그       | 설명           |
| ------------ | -------------- |
| `--json`     | 원시 JSON 출력 |
| `-h, --help` | 도움말 표시    |

## 예제

```bash

# Vercel 프로그램 표시

ossperks show vercel

# 기계 판독 가능한 JSON

ossperks show vercel --json
```
