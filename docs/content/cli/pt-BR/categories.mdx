---
title: ossperks categories
description: Lista todas as categorias de programas disponíveis e seus slugs.
---

## Uso

```
ossperks categories [opções]
```

## Opções

| Opção        | Descrição        |
| ------------ | ---------------- |
| `--json`     | Exibir JSON puro |
| `-h, --help` | Mostrar ajuda    |

## Exemplos

```bash

# Listar todas as categorias

ossperks categories

# Saída em JSON

ossperks categories --json
```

## Saída

```
CATEGORY SLUG      LABEL
hosting            Hosting & Deployment
devtools           Developer Tools
security           Security
monitoring         Monitoring & Observability
communication      Communication
ci-cd              CI/CD
testing            Testing
ai                 AI & Machine Learning
analytics          Analytics
infrastructure     Infrastructure
credentials        Credentials & Secrets
```
