---
title: ossperks list
description: Listar todos os programas de benefícios OSS disponíveis,
  opcionalmente filtrados por categoria.
---

## Uso

```
ossperks list [options]
```

**Alias:** `ossperks ls`

## Opções

| Flag                    | Descrição                                |
| ----------------------- | ---------------------------------------- |
| `-c, --category <id>` | Filtrar por id de categoria (ex: `hosting`) |
| `--json`                | Exibir JSON bruto                        |
| `-h, --help`            | Mostrar ajuda                            |

## Exemplos

```bash

# Listar todos os programas

ossperks list

# Filtrar apenas programas de hospedagem

ossperks list --category hosting

# Saída JSON legível por máquina

ossperks list --json
```
