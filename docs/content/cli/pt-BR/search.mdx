---
title: ossperks search
description: Buscar programas de benefícios OSS por palavra-chave em nomes,
  descrições e provedores.
---

## Uso

```
ossperks search [consulta] [opções]
```

**Apelidos:** `ossperks find`, `ossperks s`

## Argumentos

| Argumento    | Descrição                                                                                                                                   |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------- |
| `[consulta]` | Termo de busca (corresponde a nome, id, descrição, provedor, categoria, tags ou benefícios). Omita em um TTY para autocompletar interativo. |

## Opções

| Flag                    | Descrição                                |
| ----------------------- | ---------------------------------------- |
| `-c, --category <slug>` | Limitar busca a uma categoria específica |
| `--json`                | Exibir JSON bruto                        |
| `-h, --help`            | Mostrar ajuda                            |

## Exemplos

```bash

# Buscar por "hosting"

ossperks search hosting

# Busca interativa com autocompletar (TTY): digite para filtrar, escolha um programa

ossperks search

# Usando o apelido curto

ossperks s credits

# Saída JSON

ossperks search vercel --json
```
