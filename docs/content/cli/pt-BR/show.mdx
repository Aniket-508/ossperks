---
title: ossperks show
description: Exibir detalhes completos de um programa de benefícios OSS específico.
---

## Uso

```
ossperks show [id] [opções]
```

**Alias:** `ossperks info`

## Argumentos

| Argumento | Descrição                                                                 |
| -------- | ----------------------------------------------------------------------------- |
| `[id]`   | ID do programa opcional (ex.: `vercel`, `github-pages`). Omita em um TTY para escolher de uma lista. |

## Opções

| Flag         | Descrição         |
| ------------ | ----------------- |
| `--json`     | Exibir JSON bruto |
| `-h, --help` | Mostrar ajuda     |

## Exemplos

```bash

# Mostrar o programa Vercel

ossperks show vercel

# Seletor interativo (TTY): escolha um programa da lista

ossperks show

# JSON legível por máquina

ossperks show vercel --json
```
