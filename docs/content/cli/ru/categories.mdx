## Использование

```
ossperks categories [options]
```

**Псевдоним:** `ossperks cats`

## Опции

| Флаг         | Описание                    |
| ------------ | --------------------------- |
| `--json`     | Вывести необработанный JSON |
| `-h, --help` | Показать справку            |

## Примеры

```bash

# TTY: автодополнение категории, затем список программ (ИМЯ + URL).

# Не-TTY: выводит таблицу категорий (ИМЯ + КОЛИЧЕСТВО).

ossperks categories

# JSON: идентификаторы категорий для ossperks list --category <id>

ossperks categories --json
```

## Вывод

В **TTY** вы получаете интерактивное приглашение с автодополнением; после выбора категории CLI выводит таблицу с **ИМЕНЕМ** и **URL** программ в этой категории (в том же стиле, что и <DynamicLink href="/[lang]/cli/list">`ossperks list`</DynamicLink>).

В **неинтерактивной** среде:

```
13 категорий
─────────────

  ИМЯ                      КОЛИЧЕСТВО
  Хостинг и развертывание       12
  Инструменты разработки         8
  ...
```

Столбец **КОЛИЧЕСТВО** показывает число программ в категории (примерные значения).

Используйте **id** категории из `--json` с командой <DynamicLink href="/[lang]/cli/list">`ossperks list --category <id>`</DynamicLink> для фильтрации программ в скриптах.
