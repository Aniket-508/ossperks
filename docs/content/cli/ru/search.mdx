## Использование

```
ossperks search [запрос] [параметры]
```

**Псевдонимы:** `ossperks find`, `ossperks s`

## Аргументы

| Аргумент  | Описание                                                                       |
| --------- | --------------------------------------------------------------------------------- |
| `[запрос]` | Поисковый запрос (соответствует названию, идентификатору, описанию, поставщику, категории, тегам или льготам). Пропустите в TTY для интерактивного автодополнения. |

## Опции

| Флаг                    | Описание                                 |
| ----------------------- | ---------------------------------------- |
| `-c, --category <slug>` | Ограничить поиск определённой категорией |
| `--json`                | Вывести необработанный JSON              |
| `-h, --help`            | Показать справку                         |

## Примеры

```bash

# Поиск по запросу "hosting"

ossperks search hosting

# Интерактивный поиск с автодополнением (TTY): введите для фильтрации, выберите программу

ossperks search

# Использование короткого псевдонима

ossperks s credits

# Вывод в формате JSON

ossperks search vercel --json
```
