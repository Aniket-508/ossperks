## Использование

```
ossperks show [id] [options]
```

**Псевдоним:** `ossperks info`

## Аргументы

| Аргумент | Описание                                                                 |
| -------- | ----------------------------------------------------------------------------- |
| `[id]`   | Необязательный идентификатор программы (например, `vercel`, `github-pages`). Пропустите в TTY, чтобы выбрать из списка. |

## Опции

| Флаг         | Описание                    |
| ------------ | --------------------------- |
| `--json`     | Вывести необработанный JSON |
| `-h, --help` | Показать справку            |

## Примеры

```bash

# Показать программу Vercel

ossperks show vercel

# Интерактивный выбор (TTY): выберите программу из списка

ossperks show

# JSON для машинной обработки

ossperks show vercel --json
```
