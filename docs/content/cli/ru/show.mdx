## Использование

```
ossperks show <slug> [options]
```

## Аргументы

| Аргумент | Описание                                                     |
| -------- | ------------------------------------------------------------ |
| `<slug>` | Идентификатор программы (например, `vercel`, `github-pages`) |

## Опции

| Флаг         | Описание                    |
| ------------ | --------------------------- |
| `--json`     | Вывести необработанный JSON |
| `-h, --help` | Показать справку            |

## Примеры

```bash

# Показать программу Vercel

ossperks show vercel

# Машиночитаемый JSON

ossperks show vercel --json
```
