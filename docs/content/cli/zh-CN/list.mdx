---
title: ossperks list
description: 列出所有可用的开源软件福利项目，可选择按类别筛选。
---

## 用法

```
ossperks list [options]
```

**别名：** `ossperks ls`

## 选项

| 标志                    | 描述                              |
| ----------------------- | --------------------------------- |
| `-c, --category <id>` | 按类别 ID 筛选（例如 `hosting`） |
| `--json`                | 输出原始 JSON                     |
| `-h, --help`            | 显示帮助                          |

## 示例

```bash

# 列出所有项目

ossperks list

# 仅筛选托管类项目

ossperks list --category hosting

# 机器可读的 JSON 输出

ossperks list --json
```
