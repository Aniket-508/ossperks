---
title: ossperks search
description: 通过关键词在名称、描述或提供方中搜索 OSS 福利项目。
---

## 用法

```
ossperks search [query] [options]
```

**别名：** `ossperks find`、`ossperks s`

## 参数说明

| 参数  | 描述                                                                       |
| --------- | --------------------------------------------------------------------------------- |
| `[query]` | 搜索词（匹配名称、ID、描述、提供商、类别、标签或福利）。在 TTY 中省略以使用交互式自动补全。 |

## 选项

| 标志                    | 说明                 |
| ----------------------- | -------------------- |
| `-c, --category <slug>` | 限定在特定分类内搜索 |
| `--json`                | 输出原始 JSON        |
| `-h, --help`            | 显示帮助信息         |

## 示例

```bash

# 搜索“hosting”

ossperks search hosting

# 交互式自动补全搜索（TTY）：输入以筛选，选择一个项目

ossperks search

# 使用简短别名

ossperks s credits

# JSON 输出

ossperks search vercel --json
```
