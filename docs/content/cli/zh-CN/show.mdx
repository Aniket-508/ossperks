---
title: ossperks show
description: 显示特定开源福利项目的完整详细信息。
---

## 用法

```
ossperks show [id] [options]
```

**别名：** `ossperks info`

## 参数

| 参数 | 描述                                                                 |
| -------- | ----------------------------------------------------------------------------- |
| `[id]`   | 可选的项目 ID（例如 `vercel`、`github-pages`）。在 TTY 中省略以从列表中选择。 |

## 选项

| 标志         | 描述          |
| ------------ | ------------- |
| `--json`     | 输出原始 JSON |
| `-h, --help` | 显示帮助      |

## 示例

```bash

# 显示 Vercel 程序

ossperks show vercel

# 交互式选择器（TTY）：从列表中选择一个项目

ossperks show

# 机器可读的 JSON

ossperks show vercel --json
```

```
─────────────────────────────────────
  Vercel for Open Source
  by Vercel  ·  hosting
─────────────────────────────────────
```
