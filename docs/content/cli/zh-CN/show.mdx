---
title: ossperks show
description: 显示特定开源福利项目的完整详细信息。
---

## 用法

```
ossperks show <slug> [options]
```

## 参数

| 参数     | 描述                                        |
| -------- | ------------------------------------------- |
| `<slug>` | 程序标识符（例如 `vercel`、`github-pages`） |

## 选项

| 标志         | 描述          |
| ------------ | ------------- |
| `--json`     | 输出原始 JSON |
| `-h, --help` | 显示帮助      |

## 示例

```bash

# 显示 Vercel 程序

ossperks show vercel

# 机器可读的 JSON

ossperks show vercel --json
```
