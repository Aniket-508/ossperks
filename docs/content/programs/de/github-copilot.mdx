## Vorteile

- **Kostenloses GitHub Copilot Pro**: Vollständiger Zugriff auf GitHub Copilot Pro für KI-gestützte Code-Vervollständigung, Chat und Vorschläge.

## Voraussetzungen

- Muss Betreuer eines beliebten Open-Source-Repositorys auf GitHub sein.
- Das Repository muss die Popularitäts-/Aktivitätsschwellenwerte von GitHub erfüllen.

## So bewerben Sie sich

1. Berechtigte Betreuer werden automatisch von GitHub identifiziert.
2. Aktivierung über die GitHub Copilot-Einstellungen.
