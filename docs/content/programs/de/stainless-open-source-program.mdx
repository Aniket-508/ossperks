- **Website**: [https://stainless.com/pricing](https://stainless.com/pricing)

## Vorteile

- **SDK-Generierung**: Bis zu 5 Generatoren für TypeScript-, Python-, Go-, Java-, Kotlin-, Ruby- und C#-SDKs kostenlos.
- **Dokumentations-Websites**: Automatisch generierte API-Dokumentation und MCP-Server aus Ihrer OpenAPI-Spezifikation.
- **Vorschau-Builds**: 100 Vorschau-Builds pro Monat mit einem Limit von 25 pro Tag zum Testen von SDK-Änderungen.

## Voraussetzungen

- Das Projekt muss über eine öffentliche OpenAPI-Spezifikation verfügen.
- Muss unter einer anerkannten Open-Source-Lizenz öffentlich verfügbar sein.

## So bewerben Sie sich

1. Registrieren Sie sich auf stainless.com mit Ihrem GitHub-Konto.
2. Verbinden Sie Ihre OpenAPI-Spezifikation und konfigurieren Sie Ihre Generatoren im kostenlosen Hobby-Plan.
