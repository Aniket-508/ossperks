- **Anbieter**: Unstatus
- **Kategorie**: Überwachung
- **Website**: [https://unstatus.app/oss](https://unstatus.app/oss)

## Vorteile

- **6 Monate des Scale-Plans**: 49 $/Monat in Credits

## Teilnahmeberechtigung

- Muss ein aktives Open-Source-Projekt sein.
- Muss begründen, warum das Projekt qualifiziert ist.

## So bewerben Sie sich

1. Reichen Sie das Bewerbungsformular über die Unstatus-OSS-Programmseite ein.
2. Unstatus prüft die Bewerbung auf Teilnahmeberechtigung.
