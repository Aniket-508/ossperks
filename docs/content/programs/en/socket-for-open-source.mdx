## Perks

- **Dependency Scanning**: Full dependency scanning across your project.
- **Real-time Alerts**: Real-time alerts for malicious packages in your dependency tree.
- **PR Security Checks**: Check every PR to ensure no malicious dependencies are added, including those from outside contributors.

## Eligibility

- Must be an open source maintainer.

1. Send an email to support[at]socket[dot]dev to get set up.
