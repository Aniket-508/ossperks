- **Proveedor**: Argos
- **Categoría**: testing
- **Sitio web**: [https://argos-ci.com/docs/open-source](https://argos-ci.com/docs/open-source)
- **Solicitar**: [mailto:contact@argos-ci.com](mailto:contact@argos-ci.com)

## Beneficios

- **Plataforma gratuita de pruebas visuales**: Acceso completo a las pruebas de regresión visual de Argos para repositorios de código abierto, que incluyen comparación de capturas de pantalla, detección de diferencias y flujos de trabajo de revisión.
- **Soporte de integración**: Funciona con Playwright, Storybook, Cypress, WebdriverIO, Puppeteer, Next.js, React Router y más.

## Elegibilidad

- El proyecto debe ser de código abierto.
- El proyecto no debe ser para uso comercial.
- El uso de Argos debe estar dentro de límites razonables.
- Debe incluir la insignia de Argos en el archivo README del repositorio que apunte a https://argos-ci.com.
- Debe incluir un enlace dofollow a https://argos-ci.com en al menos una página.
- Es posible que el plan gratuito ya sea suficiente; el patrocinio es para proyectos que necesitan más.

## Cómo solicitar

1. Verifica que tu proyecto cumple con todos los criterios de patrocinio.
2. Envía un correo electrónico a contact@argos-ci.com confirmando tu elegibilidad y explicando por qué el nivel gratuito es insuficiente.
3. Incluye el slug de tu equipo de Argos y un enlace a tu repositorio de código abierto.
