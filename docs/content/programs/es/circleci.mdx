---
title: CircleCI para Código Abierto
description: CircleCI proporciona generosos créditos gratuitos de compilación
  para proyectos de código abierto en el Plan Gratuito, cubriendo compilaciones
  en Linux, Arm, Docker, macOS y Windows.
---

- **Proveedor**: CircleCI
- **Categoría**: ci-cd
- **Sitio web**: [https://circleci.com/open-source/](https://circleci.com/open-source/)

## Beneficios

- **400,000 Créditos Linux/Mes**: Hasta 400,000 créditos mensuales para compilaciones en Linux, Arm y Docker.
- **30,000 Créditos macOS/Windows**: 30,000 créditos mensuales para compilaciones en macOS y Windows.

## Elegibilidad

- Debe estar en el Plan Gratuito de CircleCI.
- El repositorio debe ser público.

## Requisitos

- Habilite la configuración "Only build pull requests" para reducir compilaciones innecesarias.

## Cómo solicitarlo

1. Regístrese en el Plan Gratuito de CircleCI.
2. Mantenga su repositorio público para calificar automáticamente para los créditos de código abierto.
