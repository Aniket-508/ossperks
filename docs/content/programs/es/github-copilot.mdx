## Beneficios

- **GitHub Copilot Pro gratuito**: Acceso completo a GitHub Copilot Pro para autocompletado de código, chat y sugerencias impulsadas por IA.

## Requisitos

- Debes ser mantenedor de un repositorio de código abierto popular en GitHub.
- El repositorio debe cumplir con los umbrales de popularidad/actividad de GitHub.

## Cómo solicitarlo

1. Los mantenedores elegibles son identificados automáticamente por GitHub.
2. Actívalo a través de la configuración de GitHub Copilot.
