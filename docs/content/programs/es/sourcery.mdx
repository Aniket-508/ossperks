## Beneficios

- **Revisiones de Código con IA Gratuitas**: Funcionalidad completa de revisión de código en repositorios públicos — las mismas características que el nivel Pro de pago.
- **Integración con IDE**: Revisiones de código con IA directamente en tu IDE para proyectos de código abierto.
- **Escaneos de Seguridad**: Escaneo de seguridad para hasta 3 repositorios con frecuencia quincenal.

## Elegibilidad

- Debe ser un proyecto de código abierto con un repositorio público en GitHub o GitLab.

## Cómo solicitar

1. Regístrate en sourcery.ai con tu cuenta de GitHub o GitLab.
2. Los repositorios públicos reciben automáticamente acceso gratuito a las revisiones de código de nivel Pro.
