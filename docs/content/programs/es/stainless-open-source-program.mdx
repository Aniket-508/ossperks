- **Proveedor**: Stainless
- **Categoría**: devtools
- **Sitio web**: [https://stainless.com/pricing](https://stainless.com/pricing)

## Beneficios

- **Generación de SDK**: Hasta 5 generadores para SDK de TypeScript, Python, Go, Java, Kotlin, Ruby y C# sin costo.
- **Sitios de documentación**: Documentación de API y servidores MCP generados automáticamente desde tu especificación OpenAPI.
- **Compilaciones de vista previa**: 100 compilaciones de vista previa por mes con un límite de 25 por día para probar cambios en los SDK.

## Elegibilidad

- El proyecto debe tener una especificación OpenAPI pública.
- Debe estar disponible públicamente bajo una licencia de código abierto reconocida.

## Cómo aplicar

1. Regístrate en stainless.com con tu cuenta de GitHub.
2. Conecta tu especificación OpenAPI y configura tus generadores en el plan Hobby gratuito.
