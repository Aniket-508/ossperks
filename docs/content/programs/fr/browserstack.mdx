- **Fournisseur** : BrowserStack
- **Catégorie** : testing
- **Durée** : Adhésion à vie
- **Site web** : [https://www.browserstack.com/open-source](https://www.browserstack.com/open-source)

## Avantages

- **Tests illimités** : Tests illimités sur plus de 3 000 navigateurs et appareils mobiles réels.
- **Tous les produits BrowserStack** : Accès à Live, Automate, Percy, App Live, Screenshots et App Automate.
- **5 utilisateurs simultanés** : 5 utilisateurs simultanés et 5 parallèles pour des tests plus rapides et une collaboration en équipe.

## Éligibilité

- Doit être un projet open source (frameworks web, bibliothèques, applications web ou applications mobiles).

## Comment postuler

1. Inscrivez-vous ou connectez-vous sur la page du programme open source de BrowserStack.
