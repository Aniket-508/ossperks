- **Fournisseur** : Cal.com
- **Catégorie** : devtools
- **Site web** : [https://cal.com/help/event-types/sponsor](https://cal.com/help/event-types/sponsor)
- **Postuler** : [mailto:peer@cal.com](mailto:peer@cal.com)

## Avantages

- **Plateforme de planification gratuite** : Accès complet à l'infrastructure de planification de Cal.com au-delà de l'offre gratuite et du noyau open source pour les projets open source éligibles.
- **Investissement providentiel potentiel** : Les entreprises open source commerciales avec un chiffre d'affaires annuel récurrent inférieur à 2 M$ peuvent également recevoir un petit investissement providentiel de l'équipe de Cal.com.

## Éligibilité

- Le projet doit être open source.
- Non commercial ou une entreprise open source commerciale avec moins de 2 M$ de revenu annuel récurrent.
- Doit inclure une bannière « Réservez-nous avec Cal.com » dans le README du dépôt avec un lien vers une page de réservation Cal.com.
- Doit inclure la même bannière dans le pied de page du site web du projet.
- Doit utiliser le format de balise UTM : ?utm_source=banner&utm_campaign=oss.
- Le niveau gratuit ou le noyau open source doit être insuffisant pour les besoins du projet.

## Comment postuler

1. Assurez-vous que votre projet répond à tous les critères de parrainage.
2. Envoyez un e-mail à peer@cal.com en expliquant votre projet et en confirmant votre éligibilité.
3. Incluez l'identifiant de votre équipe Cal.com et un lien vers votre dépôt open source.
4. Les demandes sont examinées mensuellement.
