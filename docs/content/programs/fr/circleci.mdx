---
title: CircleCI pour l'Open Source
description: CircleCI offre des crédits de build gratuits généreux pour les
  projets open source sur le forfait gratuit, couvrant les builds Linux, Arm,
  Docker, macOS et Windows.
---

- **Fournisseur** : CircleCI
- **Catégorie** : ci-cd
- **Site web** : [https://circleci.com/open-source/](https://circleci.com/open-source/)

## Avantages

- **400 000 crédits Linux/mois** : Jusqu'à 400 000 crédits par mois pour les builds Linux, Arm et Docker.
- **30 000 crédits macOS/Windows** : 30 000 crédits par mois pour les builds macOS et Windows.

## Éligibilité

- Doit être sur le forfait gratuit de CircleCI.
- Le dépôt doit être public.

## Prérequis

- Activer le paramètre « Only build pull requests » pour réduire les builds inutiles.

## Comment postuler

1. Inscrivez-vous au forfait gratuit de CircleCI.
2. Gardez votre dépôt public pour être automatiquement éligible aux crédits open source.
