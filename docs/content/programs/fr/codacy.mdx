## Avantages

- **Forfait Team gratuit** : Accès complet à la plateforme cloud sans frais pour les projets open source, incluant les tableaux de bord d'équipe, les portes de qualité et l'intégration des PR.
- **Analyse de sécurité** : SAST, détection de secrets et analyse des vulnérabilités de dépendances pour les 49 langages pris en charge.
- **Suivi de la couverture de code** : Suivez les métriques de couverture de tests avec des rapports en ligne et des portes de couverture sur les pull requests.

## Éligibilité

- Doit être un projet open source avec un dépôt public.

## Comment postuler

1. Inscrivez-vous sur codacy.com avec votre compte GitHub, GitLab ou Bitbucket.
2. Les dépôts open source sont automatiquement éligibles au forfait Team gratuit.
