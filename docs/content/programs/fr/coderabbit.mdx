## Avantages

- **Revues de code IA gratuites** : Accès complet à la plateforme de revue de code IA de CodeRabbit pour tous les projets open source sans frais.
- **Financement par parrainage** : Soutien financier dans le cadre de l'engagement de 1 million de dollars de CodeRabbit envers l'open source pour les projets éligibles.
- **Filtrage des PR indésirables** : Outils basés sur l'IA pour filtrer les pull requests de faible qualité et générées par IA, réduisant ainsi la charge de travail des mainteneurs.

## Éligibilité

- Doit être un projet open source avec un dépôt accessible au public.
- Le dépôt doit être hébergé sur GitHub ou GitLab.
- Doit démontrer un développement actif et un engagement communautaire.

## Comment postuler

1. Remplissez le formulaire de candidature au programme OSS de CodeRabbit.
2. CodeRabbit examine la candidature et se réserve le droit d'approuver ou de refuser le parrainage.
