## Avantages

- **GitHub Copilot Pro gratuit** : Accès complet à GitHub Copilot Pro pour la complétion de code assistée par IA, le chat et les suggestions.

## Éligibilité

- Doit être mainteneur d'un dépôt open source populaire sur GitHub.
- Le dépôt doit répondre aux seuils de popularité/activité de GitHub.

## Comment postuler

1. Les mainteneurs éligibles sont automatiquement identifiés par GitHub.
2. Activez via les paramètres de GitHub Copilot.
