- **Fournisseur** : GitLab
- **Catégorie** : devtools
- **Site web** : [https://about.gitlab.com/solutions/open-source/](https://about.gitlab.com/solutions/open-source/)
- **Candidater** : [https://about.gitlab.com/solutions/open-source/join/](https://about.gitlab.com/solutions/open-source/join/)

## Avantages

- **Fonctionnalités GitLab Ultimate** : Accès à toutes les fonctionnalités GitLab Ultimate, y compris CI/CD avancé, analyse de sécurité et outils de conformité.
- **50 000 minutes de calcul** : 50 000 minutes de calcul par mois pour les pipelines CI/CD.

## Éligibilité

- Doit être un projet open source.
- Le projet doit être accessible au public.
- Doit encourager la collaboration ouverte et la réutilisation.

## Comment postuler

1. Rejoignez le programme GitLab pour l'Open Source via la page de candidature.
2. Posez vos questions dans la catégorie open source du forum GitLab.
