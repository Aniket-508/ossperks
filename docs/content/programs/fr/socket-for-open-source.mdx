## Avantages

- **Analyse des dépendances** : Analyse complète des dépendances dans l'ensemble de votre projet.
- **Alertes en temps réel** : Alertes en temps réel pour les packages malveillants dans votre arbre de dépendances.
- **Vérifications de sécurité des PR** : Vérifiez chaque PR pour vous assurer qu'aucune dépendance malveillante n'est ajoutée, y compris celles provenant de contributeurs externes.

## Éligibilité

- Vous devez être un mainteneur de logiciel open source.

1. Envoyez un e-mail à support[arobase]socket[point]dev pour vous inscrire.
