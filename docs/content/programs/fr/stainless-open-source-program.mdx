- **Fournisseur** : Stainless
- **Catégorie** : devtools
- **Site web** : [https://stainless.com/pricing](https://stainless.com/pricing)

## Avantages

- **Génération de SDK** : Jusqu'à 5 générateurs pour les SDK TypeScript, Python, Go, Java, Kotlin, Ruby et C# sans frais.
- **Sites de documentation** : Documentation API et serveurs MCP générés automatiquement à partir de votre spécification OpenAPI.
- **Builds de prévisualisation** : 100 builds de prévisualisation par mois avec une limite de 25/jour pour tester les modifications de SDK.

## Éligibilité

- Le projet doit disposer d'une spécification OpenAPI publique.
- Doit être disponible publiquement sous une licence open-source reconnue.

## Comment postuler

1. Inscrivez-vous sur stainless.com avec votre compte GitHub.
2. Connectez votre spécification OpenAPI et configurez vos générateurs sur le plan Hobby gratuit.
