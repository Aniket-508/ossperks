---
title: Programme OSS Unstatus
description: Forfait Unstatus Scale pour les projets open source éligibles.
---

- **Fournisseur** : Unstatus
- **Catégorie** : surveillance
- **Site web** : [https://unstatus.app/oss](https://unstatus.app/oss)

## Avantages

- **6 mois du forfait Scale** : 49 $/mois en crédits

## Éligibilité

- Doit être un projet open source actif.
- Doit justifier pourquoi le projet est éligible.

## Comment postuler

1. Soumettez le formulaire de candidature via la page du programme Unstatus OSS.
2. Unstatus examine la candidature pour vérifier l'éligibilité.
