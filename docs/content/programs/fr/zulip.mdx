## Avantages

- **Zulip Cloud Standard gratuit** : Hébergement complet Zulip Cloud Standard sans frais pour les projets open source.
- **Historique complet des messages** : Accédez à l'intégralité de l'historique des messages de votre organisation avec recherche en texte intégral.
- **Option d'accès public** : N'importe qui peut consulter, parcourir et rechercher le contenu public de votre organisation sans compte.
- **Intégrations** : Intégrations natives pour GitHub, Jira, Sentry et des centaines d'autres outils.

## Éligibilité

- Doit être un projet open source.

## Comment postuler

1. Créez une organisation Zulip.
2. Demandez un parrainage via la page de parrainage Zulip.
