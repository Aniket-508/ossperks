## 特典

- **無料のGitHub Copilot Pro**: AIを活用したコード補完、チャット、提案機能を備えたGitHub Copilot Proへのフルアクセス。

## 利用資格

- GitHub上で人気のあるオープンソースリポジトリのメンテナーであること。
- リポジトリがGitHubの人気度・活動基準を満たしていること。

## 申請方法

1. 対象となるメンテナーはGitHubによって自動的に識別されます。
2. GitHub Copilotの設定から有効化してください。
