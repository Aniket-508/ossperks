- **プロバイダー**: Stainless
- **カテゴリ**: devtools
- **ウェブサイト**: [https://stainless.com/pricing](https://stainless.com/pricing)

## 特典

- **SDK生成**: TypeScript、Python、Go、Java、Kotlin、Ruby、C#のSDKジェネレーターを最大5つまで無料で利用可能。
- **ドキュメントサイト**: OpenAPI仕様から自動生成されるAPIドキュメントとMCPサーバー。
- **プレビュービルド**: 月間100回のプレビュービルド(1日あたり25回の制限付き)でSDK変更のテストが可能。

## 利用資格

- プロジェクトは公開されたOpenAPI仕様を持っている必要があります。
- 認定されたオープンソースライセンスの下で公開されている必要があります。

## 申し込み方法

1. GitHubアカウントでstainless.comにサインアップします。
2. OpenAPI仕様を接続し、無料のHobbyプランでジェネレーターを設定します。
