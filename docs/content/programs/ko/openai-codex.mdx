## 혜택

- **6개월 ChatGPT Pro**: 코딩, 이슈 분류, 리뷰 및 유지관리자 워크플로를 위한 Codex 액세스가 포함된 무료 ChatGPT Pro 구독.
- **API 크레딧**: PR 리뷰, 자동화, 릴리스 워크플로 또는 핵심 오픈소스 작업을 위한 Codex Open Source Fund를 통한 크레딧.
- **Codex Security**: 적격 저장소에 대한 심층 코드 보안 스캔 및 감사를 위한 Codex Security 조건부 액세스.

## 자격 요건

- GitHub 스타 수나 월간 다운로드 기준과 같은 엄격한 요구 사항은 없습니다.
- 오픈소스 프로젝트의 주요 또는 핵심 유지관리자여야 합니다.
- 저장소 사용량, 생태계 중요성 및 활발한 유지관리를 입증해야 합니다.
- 유효한 ChatGPT 계정이 있어야 합니다.

## 신청 방법

1. OpenAI의 공식 Codex for OSS 양식을 통해 제출합니다.
2. 신청서는 수시로 검토됩니다.
3. 선정된 신청자에게는 이메일로 통보됩니다.
