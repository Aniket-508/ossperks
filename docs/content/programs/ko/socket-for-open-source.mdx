## 혜택

- **종속성 스캔**: 프로젝트 전체에 걸친 완전한 종속성 스캔을 제공합니다.
- **실시간 알림**: 종속성 트리에서 악성 패키지에 대한 실시간 알림을 제공합니다.
- **PR 보안 검사**: 외부 기여자의 종속성을 포함하여 악성 종속성이 추가되지 않도록 모든 PR을 검사합니다.

## 자격 요건
