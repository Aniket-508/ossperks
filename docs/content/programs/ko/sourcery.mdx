## 혜택

- **무료 AI 코드 리뷰**: 공개 저장소에서 전체 코드 리뷰 기능 제공 — 유료 Pro 등급과 동일한 기능.
- **IDE 통합**: 오픈 소스 프로젝트를 위한 IDE 내 AI 기반 코드 리뷰.
- **보안 스캔**: 최대 3개의 저장소에 대해 격주 주기로 보안 스캔 제공.

## 자격 요건

- GitHub 또는 GitLab에 공개 저장소가 있는 오픈 소스 프로젝트여야 합니다.

## 신청 방법

1. GitHub 또는 GitLab 계정으로 sourcery.ai에 가입하세요.
2. 공개 저장소는 자동으로 Pro 등급 코드 리뷰에 무료로 액세스할 수 있습니다.
