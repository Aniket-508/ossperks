- **제공자**: Stainless
- **카테고리**: devtools
- **웹사이트**: [https://stainless.com/pricing](https://stainless.com/pricing)

## 혜택

- **SDK 생성**: TypeScript, Python, Go, Java, Kotlin, Ruby, C# SDK를 위한 최대 5개의 생성기를 무료로 제공합니다.
- **문서 사이트**: OpenAPI 스펙으로부터 자동 생성되는 API 문서 및 MCP 서버.
- **프리뷰 빌드**: SDK 변경 사항 테스트를 위해 월 100회의 프리뷰 빌드 제공 (일일 25회 제한).

## 자격 요건

- 프로젝트는 공개 OpenAPI 스펙을 보유해야 합니다.
- 인정된 오픈소스 라이선스 하에 공개적으로 사용 가능해야 합니다.
