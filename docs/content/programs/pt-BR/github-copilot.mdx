## Benefícios

- **GitHub Copilot Pro Gratuito**: Acesso completo ao GitHub Copilot Pro para conclusão de código, chat e sugestões assistidas por IA.

## Elegibilidade

- Deve ser mantenedor de um repositório de código aberto popular no GitHub.
- O repositório deve atender aos limites de popularidade/atividade do GitHub.

## Como se candidatar

1. Mantenedores elegíveis são automaticamente identificados pelo GitHub.
2. Ative através das configurações do GitHub Copilot.
