- **Provedor**: Sentry
- **Categoria**: monitoramento
- **Duração**: Sem limite de prazo
- **Site**: [https://sentry.io/for/open-source/](https://sentry.io/for/open-source/)

## Vantagens

- **5 Milhões de Erros**: Rastreie até 5 milhões de erros gratuitamente.
- **100 Milhões de Transações**: Monitoramento de desempenho com até 100 milhões de transações.
- **10 GB de Anexos**: Armazene até 10 GB de anexos de erros.
- **100.000 Replays de Sessão**: Reproduza sessões de usuários para depurar problemas.

## Elegibilidade

- Deve ser um projeto de código aberto ativo.
- O projeto deve usar uma licença de código aberto reconhecida.
