## Benefícios

- **Revisões de Código com IA Gratuitas**: Funcionalidade completa de revisão de código em repositórios públicos — mesmos recursos do plano pago Pro.
- **Integração com IDE**: Revisões de código com IA diretamente na sua IDE para projetos de código aberto.
- **Verificações de Segurança**: Verificação de segurança para até 3 repositórios com frequência quinzenal.

## Elegibilidade

- Deve ser um projeto de código aberto com um repositório público no GitHub ou GitLab.

## Como se candidatar

1. Cadastre-se em sourcery.ai com sua conta do GitHub ou GitLab.
2. Repositórios públicos recebem automaticamente acesso gratuito às revisões de código do nível Pro.
