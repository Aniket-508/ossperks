- **Fornecedor**: Stainless
- **Categoria**: devtools
- **Site**: [https://stainless.com/pricing](https://stainless.com/pricing)

## Benefícios

- **Geração de SDK**: Até 5 geradores para SDKs em TypeScript, Python, Go, Java, Kotlin, Ruby e C# sem custo.
- **Sites de Documentação**: Documentação de API e servidores MCP gerados automaticamente a partir da sua especificação OpenAPI.
- **Builds de Prévia**: 100 builds de prévia por mês com limite de 25/dia para testar alterações nos SDKs.

## Elegibilidade

- O projeto deve ter uma especificação OpenAPI pública.
- Deve estar disponível publicamente sob uma licença de código aberto reconhecida.

## Como se candidatar

1. Cadastre-se em stainless.com com sua conta do GitHub.
2. Conecte sua especificação OpenAPI e configure seus geradores no plano gratuito Hobby.
