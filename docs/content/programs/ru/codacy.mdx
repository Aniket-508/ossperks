## Преимущества

- **Бесплатный командный план**: Полный доступ к облачной платформе без каких-либо затрат для проектов с открытым исходным кодом, включая командные панели, контроль качества и интеграцию с pull request.
- **Сканирование безопасности**: SAST, обнаружение секретов и сканирование уязвимостей зависимостей для всех 49 поддерживаемых языков.
- **Отслеживание покрытия кода**: Отслеживание метрик покрытия тестами с встроенной отчётностью и контролем покрытия в pull request.

## Требования

- Должен быть проектом с открытым исходным кодом с публичным репозиторием.

## Как подать заявку

1. Зарегистрируйтесь на codacy.com с помощью учётной записи GitHub, GitLab или Bitbucket.
2. Репозитории с открытым исходным кодом автоматически получают право на бесплатный командный план.
