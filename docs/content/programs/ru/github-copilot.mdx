## Преимущества

- **Бесплатный GitHub Copilot Pro**: Полный доступ к GitHub Copilot Pro для автодополнения кода с помощью ИИ, чата и подсказок.

## Условия участия

- Необходимо быть мейнтейнером популярного репозитория с открытым исходным кодом на GitHub.
- Репозиторий должен соответствовать пороговым значениям популярности и активности GitHub.

## Как подать заявку

1. Подходящие мейнтейнеры автоматически определяются GitHub.
2. Активируйте через настройки GitHub Copilot.
