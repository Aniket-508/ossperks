- **Провайдер**: PostHog
- **Категория**: аналитика
- **Веб-сайт**: [https://posthog.com/startups](https://posthog.com/startups)

## Преимущества

- **Кредиты PostHog**: $50 000 в кредитах, покрывающих все продукты, включая аналитику продукта, воспроизведение сессий, флаги функций, эксперименты и отслеживание ошибок.
- **Отмена срока действия кредитов**: Стандартный 12-месячный срок действия кредитов отменяется для подходящих OSS-проектов по запросу.

## Право на участие

- Должен быть проектом с открытым исходным кодом без корпоративной поддержки.
- Годовая выручка должна быть менее $200 000.

## Как подать заявку

1. Зарегистрируйтесь в PostHog Cloud на сайте posthog.com (кредитная карта не требуется).
2. Подайте заявку через программу PostHog для стартапов.
3. Свяжитесь со службой поддержки PostHog, чтобы отменить 12-месячный срок действия кредитов.
