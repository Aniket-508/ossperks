- **Провайдер**: Sentry
- **Категория**: мониторинг
- **Срок действия**: Без ограничений по времени
- **Веб-сайт**: [https://sentry.io/for/open-source/](https://sentry.io/for/open-source/)

## Преимущества

- **5 миллионов ошибок**: Отслеживайте до 5 миллионов ошибок бесплатно.
- **100 миллионов транзакций**: Мониторинг производительности с поддержкой до 100 миллионов транзакций.
- **10 ГБ вложений**: Храните до 10 ГБ вложений с ошибками.
- **100 000 повторов сеансов**: Воспроизводите пользовательские сеансы для отладки проблем.

## Требования

- Должен быть активным проектом с открытым исходным кодом.
- Проект должен использовать признанную лицензию с открытым исходным кодом.
