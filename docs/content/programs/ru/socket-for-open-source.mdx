## Преимущества

- **Сканирование зависимостей**: Полное сканирование зависимостей всего проекта.
- **Оповещения в реальном времени**: Оповещения в реальном времени о вредоносных пакетах в дереве зависимостей.
- **Проверки безопасности PR**: Проверка каждого пул-реквеста для предотвращения добавления вредоносных зависимостей, включая те, что исходят от внешних участников.

## Требования

- Необходимо быть разработчиком открытого исходного кода.
