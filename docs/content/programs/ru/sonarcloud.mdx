## Преимущества

- **Бесплатный статический анализ**: Анализируйте публичные проекты бесплатно с поддержкой сканирования качества кода и безопасности.
- **Неограниченное количество участников**: Масштабируйте команду без ограничений — приглашайте всех участников бесплатно.
- **Анализ feature-веток**: Выявляйте проблемы до их попадания в основную ветку с помощью анализа feature-веток.
- **Настраиваемые стандарты качества**: Определяйте собственные стандарты качества кода и безопасности.
- **Синхронизация участников GitHub**: Удобно управляйте участниками с автоматической синхронизацией с GitHub.

## Требования

- Проект должен быть публично доступным open-source проектом.
- Репозиторий должен быть публичным на GitHub.

## Как подать заявку

1. Зарегистрируйтесь в SonarQube Cloud с вашей организацией GitHub.
2. Публичные проекты автоматически получают право на бесплатный OSS-план.
