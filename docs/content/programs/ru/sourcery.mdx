## Преимущества

- **Бесплатные обзоры кода с помощью ИИ**: Полный функционал проверки кода в публичных репозиториях — те же возможности, что и в платном тарифе Pro.
- **Интеграция с IDE**: Обзоры кода на базе ИИ прямо в вашей IDE для проектов с открытым исходным кодом.
- **Сканирование безопасности**: Проверка безопасности для до 3 репозиториев с периодичностью раз в две недели.

## Требования для участия

- Должен быть проектом с открытым исходным кодом с публичным репозиторием на GitHub или GitLab.

## Как подать заявку

1. Зарегистрируйтесь на sourcery.ai с помощью вашего аккаунта GitHub или GitLab.
2. Публичные репозитории автоматически получают бесплатный доступ к обзорам кода уровня Pro.
