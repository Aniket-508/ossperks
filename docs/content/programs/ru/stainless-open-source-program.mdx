- **Провайдер**: Stainless
- **Категория**: devtools
- **Веб-сайт**: [https://stainless.com/pricing](https://stainless.com/pricing)

## Преимущества

- **Генерация SDK**: До 5 генераторов для SDK на TypeScript, Python, Go, Java, Kotlin, Ruby и C# бесплатно.
- **Сайты документации**: Автоматически создаваемая документация API и MCP-серверы из вашей спецификации OpenAPI.
- **Предварительные сборки**: 100 предварительных сборок в месяц с ограничением 25 в день для тестирования изменений SDK.

## Требования

- Проект должен иметь публичную спецификацию OpenAPI.
- Должен быть общедоступным под признанной лицензией с открытым исходным кодом.

## Как подать заявку

1. Зарегистрируйтесь на stainless.com с помощью вашей учетной записи GitHub.
2. Подключите вашу спецификацию OpenAPI и настройте генераторы на бесплатном плане Hobby.
