- **Провайдер**: Vercel
- **Категория**: хостинг
- **Длительность**: 12 месяцев
- **Веб-сайт**: [https://vercel.com/open-source-program](https://vercel.com/open-source-program)

## Преимущества

- **$3,600 в виде кредитов платформы**: $3,600 в виде кредитов платформы Vercel, распределенных на 12 месяцев.
- **Стартовый пакет для OSS**: Кредиты от сторонних сервисов для развития вашего проекта.
- **Поддержка сообщества**: Приоритетная поддержка сообщества и рекомендации от Vercel.

## Требования

- Проект должен быть открытым и активно разрабатываться и поддерживаться.
- Проект должен быть размещен (или планируется разместить) на Vercel.
- Проект должен демонстрировать измеримое влияние или потенциал роста.
- Проект должен следовать Кодексу поведения.
- Кредиты должны использоваться исключительно для работы с открытым исходным кодом.

## Как подать заявку

1. Подайте заявку через страницу программы Vercel OSS, когда прием заявок открыт.
2. Заявки рассматриваются сезонными группами.
