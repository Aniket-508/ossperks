## Преимущества

- **Бесплатный Zulip Cloud Standard**: Полный хостинг Zulip Cloud Standard бесплатно для проектов с открытым исходным кодом.
- **Полная история сообщений**: Доступ ко всей истории сообщений вашей организации с полнотекстовым поиском.
- **Опция публичного доступа**: Любой может просматривать, изучать и искать публичный контент вашей организации без учетной записи.
- **Интеграции**: Встроенные интеграции для GitHub, Jira, Sentry и сотен других инструментов.

## Требования

- Должен быть проектом с открытым исходным кодом.
