- **提供商**：Argos
- **类别**：测试
- **网站**：[https://argos-ci.com/docs/open-source](https://argos-ci.com/docs/open-source)
- **申请**：[mailto:contact@argos-ci.com](mailto:contact@argos-ci.com)

## 权益

- **免费视觉测试平台**：为开源仓库提供 Argos 视觉回归测试的完整访问权限，涵盖截图比对、差异检测和审查工作流。
- **集成支持**：支持 Playwright、Storybook、Cypress、WebdriverIO、Puppeteer、Next.js、React Router 等工具。

## 资格要求

- 项目必须是开源的。
- 项目不得用于商业用途。
- Argos 的使用必须在合理使用范围内。
- 必须在仓库 README 中包含指向 https://argos-ci.com 的 Argos 徽章。
- 必须在至少一个页面上包含指向 https://argos-ci.com 的 dofollow 链接。
- 免费计划可能已经足够——赞助是为需要更多资源的项目准备的。

## 如何申请

1. 验证您的项目符合所有赞助标准。
2. 发送电子邮件至 contact@argos-ci.com，确认资格并说明为什么免费套餐不足以满足需求。
3. 包含您的 Argos 团队标识（team slug）和开源仓库链接。
