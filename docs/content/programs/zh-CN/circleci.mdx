description: CircleCI 为免费计划中的开源项目提供慷慨的免费构建积分,涵盖 Linux、Arm、Docker、macOS 和 Windows 构建。
---

- **提供商**：CircleCI
- **类别**：ci-cd
- **网站**：[https://circleci.com/open-source/](https://circleci.com/open-source/)

## 优势

- **每月 400,000 Linux 积分**：每月最多 400,000 积分，用于 Linux、Arm 和 Docker 构建。
- **30,000 macOS/Windows 积分**：每月 30,000 积分，用于 macOS 和 Windows 构建。

## 资格要求

- 必须使用 CircleCI 的免费计划。
- 代码仓库必须是公开的。

## 要求

- 启用"仅构建拉取请求"设置，以减少不必要的构建。

## 申请方式

1. 注册 CircleCI 的免费计划。
2. 保持代码仓库公开，即可自动获得开源积分资格。

## 标签

`ci-cd`、`builds`、`automation`、`continuous-integration`
