## 福利

- **免费的 GitHub Copilot Pro**：全面使用 GitHub Copilot Pro 的 AI 代码补全、聊天和建议功能。

## 资格要求

- 必须是 GitHub 上热门开源仓库的维护者。
- 仓库必须达到 GitHub 的热门度/活跃度门槛。

## 申请方式

1. GitHub 会自动识别符合条件的维护者。
2. 通过 GitHub Copilot 设置进行激活。
