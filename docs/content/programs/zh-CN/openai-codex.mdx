- **提供商**：OpenAI
- **类别**：ai
- **期限**：6 个月
- **网站**：[https://openai.com/form/codex-for-oss/](https://openai.com/form/codex-for-oss/)

## 福利

- **6 个月 ChatGPT Pro**：免费 ChatGPT Pro 订阅，包含 Codex 访问权限，可用于编码、分类、审查和维护者工作流程。
- **API 额度**：通过 Codex 开源基金提供的额度，可用于 PR 审查、自动化、发布工作流程或核心开源工作。
- **Codex 安全**：有条件地访问 Codex Security，对符合条件的仓库进行更深入的代码安全扫描和审计。

## 资格要求

- 没有硬性要求，如 GitHub 星标数或月下载量阈值。
- 必须是开源项目的主要或核心维护者。
- 必须证明仓库的使用情况、生态系统重要性和活跃维护。
- 必须拥有有效的 ChatGPT 账户。

## 如何申请

1. 通过 OpenAI 官方的 Codex for OSS 表单提交申请。
2. 申请将按滚动方式审核。
3. 入选的申请者将通过电子邮件收到通知。
