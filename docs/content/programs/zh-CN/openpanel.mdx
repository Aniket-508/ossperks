- **提供商**：OpenPanel
- **类别**：分析
- **网站**：[https://openpanel.dev/open-source](https://openpanel.dev/open-source)

## 福利

- **每月250万次事件**：每月最多跟踪250万次页面浏览、用户操作和自定义事件。
- **完整功能访问**：与付费计划相同的功能：漏斗分析、留存分析、自定义仪表板和实时分析。
- **无限团队成员**：邀请您的整个贡献者团队协作分析数据。
- **优先支持**：为开源维护者提供专属帮助，响应时间更快。

## 资格要求

- 必须使用 OSI 批准的开源许可证（MIT、Apache、GPL 等）。
- 代码必须在 GitHub、GitLab 或类似平台上公开可用。
- 必须展示活跃开发和不断增长的社区的证据。
- 主要目的应为非商业性质（商业性开源项目可能会被考虑）。

## 要求

- 在您的网站或 README 中添加指向 OpenPanel 的反向链接（例如："分析由 OpenPanel 提供支持"）。
- 可选择显示 OpenPanel 访客计数小部件。

## 如何申请

1. 发送电子邮件至 oss@openpanel.dev，附上您的项目详情。
2. 包括您的项目网址、许可证类型和简要描述。
3. OpenPanel 会在几个工作日内审核并回复。
