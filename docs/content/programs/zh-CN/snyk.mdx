- **提供商**：Snyk
- **类别**：安全
- **网站**：[https://snyk.io/open-source/](https://snyk.io/open-source/)

## 权益

- **完整企业版许可证**：完整的 Snyk 企业版访问权限，无使用限制,适用于开源项目。
- **所有最新功能**：无限制使用所有 Snyk 最新功能。
- **社区支持**：通过 Snyk 的 Discord 服务器提供社区支持。
- **安全咨询协助**：专门的安全咨询协助，帮助设置和集成。

## 资格要求

- 项目不得由任何企业实体支持。
- 项目不得寻求通过商业化、授权或服务获利。
- 软件必须使用 OSI 批准的开源许可证。

## 要求

- 在项目的 README 和网站中包含指向 Snyk 网站的链接。
- 允许 Snyk 在其网站上使用、复制和展示项目的徽标。
