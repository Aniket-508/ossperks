## 权益

- **依赖扫描**：对您的项目进行全面的依赖扫描。
- **实时警报**：针对依赖树中的恶意软件包提供实时警报。
- **PR 安全检查**：检查每个 PR 以确保不会添加恶意依赖项，包括来自外部贡献者的依赖项。

## 资格要求

- 必须是开源维护者。
