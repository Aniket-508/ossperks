## 权益

- **免费 AI 代码审查**：公共仓库享有完整的代码审查功能——与付费专业版相同的功能。
- **IDE 集成**：开源项目可直接在 IDE 中使用 AI 驱动的代码审查。
- **安全扫描**：最多 3 个仓库可享有双周频率的安全扫描。

## 资格要求

- 必须是在 GitHub 或 GitLab 上拥有公共仓库的开源项目。

## 申请方式

1. 使用您的 GitHub 或 GitLab 账户在 sourcery.ai 注册。
2. 公共仓库将自动获得专业版代码审查的免费访问权限。
