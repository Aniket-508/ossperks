- **提供商**：Stainless
- **类别**：开发工具
- **网站**：[https://stainless.com/pricing](https://stainless.com/pricing)

## 权益

- **SDK 生成**：免费提供最多 5 个生成器，支持 TypeScript、Python、Go、Java、Kotlin、Ruby 和 C# SDK。
- **文档站点**：根据您的 OpenAPI 规范自动生成 API 文档和 MCP 服务器。
- **预览构建**：每月 100 次预览构建，每日速率限制为 25 次，用于测试 SDK 更改。

## 资格要求

- 项目必须拥有公开的 OpenAPI 规范。
- 必须在公认的开源许可证下公开提供。

## 如何申请

1. 使用您的 GitHub 账户在 stainless.com 注册。
2. 连接您的 OpenAPI 规范，并在免费的 Hobby 计划上配置生成器。
